# Бот автоответчик "Igra glagolov"

Проект учебного курса Devman.  
Реализация чат-ботов службы поддержки онлайн-издательства **«Игра глаголов»** для Telegram и ВКонтакте.

Боты распознают сообщения пользователей с помощью DialogFlow и автоматически отвечают на типовые вопросы.  
Если бот не понимает сообщение, оно передаётся операторам поддержки (для ВКонтакте бот в этом случае не отвечает).

---

## Функциональность

### Telegram бот
- Принимает сообщения от пользователей
- Отправляет текст пользователя в DialogFlow
- Отвечает пользователю фразами из DialogFlow
- Отправляет уведомление в Telegram при запуске бота
- Отправляет уведомление с текстом ошибки при падении бота

  ![ezgif com-video-to-gif-converter](https://github.com/user-attachments/assets/19e0e81e-a691-44e6-a67e-60d5a9d9123d)


### VK бот
- Работает от имени группы ВКонтакте
- Получает входящие сообщения пользователей
- Отвечает только если DialogFlow распознал запрос
- Если DialogFlow не понял сообщение — бот молчит (сообщение уходит операторам)

## Переменные окружения

Для работы проекта необходимо создать файл `.env` со следующими переменными:

```env
TG_BOT_API=токен_telegram_бота
TG_USER_ID=telegram_id_для_получения_логов
TG_DEBUG_TOKEN=токен_логгер_бота

PROJECT_ID=google_dialogflow_project_id
GOOGLE_APPLICATION_CREDENTIALS=/absolute/path/to/credentials.json

VK_GROUP_TOKEN=токен_группы_вконтакте
```

## Запуск проекта локально

Создать виртуальное окружение и активировать его:
```
python -m venv venv
source venv/bin/activate
```

### Установить зависимости
```
pip install -r requirements.txt
```

Создать файл .env и заполнить переменные окружения

### Запустить Telegram бота:
```
python tg_bot.py
```

### Запустить VK бота:
```
python vk_bot.py
```
